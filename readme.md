# FRP Streams Library Benchmarks

Speed and Memory Benchmark of"
- Imperative
- RxJs
- XStream
- Most/Core

## Installation

```bash
yarn install
```

## Test
```bash
yarn test
```

## Speed Benchmark
```bash
yarn run speed
```
Results files:

- .tmp/data-promise.html
- .tmp/data-recursive.html

## Memory Benchmark
```bash
yarn run memory
```
